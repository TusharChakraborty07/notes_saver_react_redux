# 📝 PasteHub — Modern Paste & Notes Manager

PasteHub is a clean and responsive web application for creating, managing, editing, viewing, and searching text snippets or notes.

Built using **React**, **Redux Toolkit**, **React Router**, and **Tailwind CSS**, the app provides a smooth note-taking and paste management experience.

---

## ✨ Features

### 📄 Create Paste

Create and save text snippets instantly.

### ✏️ Edit Existing Paste

Update existing notes without creating duplicates.

### 🔍 Search Paste

Search by title or content in real time.

### 👀 View Paste

Open a dedicated page to read paste details.

### 📋 Copy Content

Copy any paste content to clipboard with one click.

### 💾 Persistent Storage

Data remains available using Redux state management.

### 📱 Responsive UI

Optimized for desktop and mobile screens.

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/TusharChakraborty07/notes_saver_react_redux/
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:

```bash
http://localhost:5173
```
